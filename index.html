Introduction

Senior Design Project

A Smart Social Distancing Regulator

It's a computer vision based application to make sure that social distancing and
mask management is regulated and keeps your social space safe!

Design Goals


Third Eye

Design Goals

Reliability

The system is resistant and is able to run and collect data 24/7.

Usability

The system is self-explanatory, user-friendly and clear in terms of display and language. It has a neat and well-organized user interface.

Maintainability

The system is open for change and is able to recieve updates whenever required. I works for long periods of time without a need for maintenance.

Customer Needs

The system is easy to set up and use for the customer.
